# Project Philosophy

BrandTrace exists to increase transparency.

Transparency creates accountability. Accountability creates trust. Trust creates better markets.

## Guiding Statement

> We document.
>
> We verify.
>
> Consumers decide.

## Neutrality Standard

BrandTrace is a factual ownership database, not a platform for political persuasion, corporate attacks, or unsupported accusations. The platform must present documented information neutrally and give consumers the context they need to make their own decisions.

## Evidence Before Claims

No unsupported claim should ever be presented as fact. Every factual statement should be verifiable whenever possible, and records should include reliable sources that support ownership relationships, acquisitions, and corporate structures.

## Information Classifications

BrandTrace must distinguish clearly between:

- **Verified Information:** Information reviewed against reliable sources and approved under the verification protocol.
- **Pending Review:** Information being evaluated by reviewers before publication as verified.
- **Community Submission:** Information submitted by users that has not yet been verified.
- **Opinion:** Subjective statements, commentary, or interpretation that must never be confused with verified fact.

## Design Questions

Before implementing any feature, ask:

1. Does this increase transparency?
2. Does this improve trust?
3. Does this empower consumers?
4. Can this information be verified?
5. Will this still make sense five years from now?

If not, redesign it.
